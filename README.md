# Critical Hit! 🎲

A pixel-art dice battle game that runs entirely in your browser. No installs, no
dependencies, no server — each game is a single self-contained HTML file. Open it
and roll.

![dice roll](dice_roll_transparent.gif)

## The games

| File | Mode |
|---|---|
| [`crystal_dice_game.html`](crystal_dice_game.html) | **Duel** — head-to-head dice battle |
| [`crystal_dice_boss_test.html`](crystal_dice_boss_test.html) | **Boss battle** — take on the big one |

## How to play

1. Download (or clone) this repo.
2. Double-click either `.html` file — it opens in your browser.
3. Everything else is on-screen.

That's it. The dice, animations, and sparkles are all hand-drawn on a `<canvas>`
at chunky pixel scale.

## Prototypes

`dice_roll_mockup.html` and `dice_roll_mockup_v2.html` are the early animation
test benches the game grew out of — kept for the archaeology.

## Origins

Built as the table-top centerpiece for a long-running roleplay campaign — the
dice needed to *feel* physical, land with weight, and occasionally betray you.
